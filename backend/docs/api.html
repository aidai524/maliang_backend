🚀 Dream API 测试工具

🔑 认证配置

🔐 认证模块

POST /v1/auth/send-code
发送短信验证码
POST /v1/auth/phone-login
手机号登录/注册（自动注册新用户）
POST /v1/auth/wechat-login
微信登录
POST /v1/auth/bind-phone
绑定手机号到当前账号
GET /v1/auth/userinfo
获取当前用户信息和统计
GET /v1/auth/check
检查登录状态

👤 用户模块

GET /v1/users/me
获取当前用户信息
GET /v1/users/me/stats
获取用户统计（收藏、生成数、配额等）

🎨 生成任务模块（本地记录）

GET /v1/jobs/quota
获取配额信息
POST /v1/jobs
创建生成任务（自动检查配额，支持锁脸、默认使用云雾API）
POST /v1/jobs (with face-lock)
创建锁脸任务（需要先创建角色并上传照片）
GET /v1/jobs
获取本地任务列表

📝 模板模块

GET /v1/templates/prompts
获取提示词模板列表
GET /v1/templates/params
获取参数模板列表
POST /v1/templates/prompts
创建提示词模板（需要 Admin Key）

🎭 角色模块

GET /v1/characters
获取我的角色列表
POST /v1/characters
创建角色
DELETE /v1/characters/:uuid
删除角色

⚙️ 管理模块

GET /v1/admin/stats
获取管理统计
GET /v1/admin/users
获取用户列表
PUT /v1/admin/users/:id/vip
设置用户 VIP
DELETE /v1/admin/users/:id/vip
取消用户 VIP
