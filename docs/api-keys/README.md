# API Keys 目录

此目录用于存储开发者和测试用的 API Token。

## 文件说明

- `developer-token.txt` - 开发者长期测试 Token (10年有效)

## ⚠️ 安全提醒

1. **不要提交到 Git** - 此目录已在 .gitignore 中排除
2. **仅用于测试** - 这些 Token 仅用于开发和测试
3. **定期更换** - 生产环境请使用正式的认证流程

## 生产环境

生产环境请使用正式的用户认证流程：
1. 用户通过手机号/微信登录
2. 获取 JWT Token
3. 使用 Token 调用 API

详见: [API 文档](../API.md)
